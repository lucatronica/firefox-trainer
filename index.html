What is this?

A game/tool for practicing sweetspotting Fox Up-B on the ledge in Melee.
You need to use a gamepad to play this. A gamecube controller may be used if it is setup as a generic gamepad.
Currently only Chrome, Firefox and Opera support the Gamepad web api.
